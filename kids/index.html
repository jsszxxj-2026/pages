⚠️ 每次训练固定流程（45分钟分配）：操场慢跑1圈热身(5min) → 主训练(30min) → 拉伸放松(5min) → 机动补水(5min)，三段缺一不可。
📋 每日详细计划
🧘 周一 · 平衡+协调~45分钟
  1. 🏃 操场慢跑1圈（5分钟）+ 甩臂10次 + 转踝各10圈 + 小弹跳10次
    ⏱️ 热身 ~5min
  2. 🧘 金鸡独立（单脚站立）— 每脚3组
    头顶网球平衡，球不掉=满分
    扶墙单脚站立，不计时
    ⏱️ ~10min（含组间休息）
  3. 🤸 跳绳 — 连续跳绳挑战
    网球5等级抛接（单手→交替→拍手→背后→墙壁反弹）
    原地蹦蹦跳或模仿甩绳
    ⏱️ ~12min
  4. 🐻 熊爬行走 — 来回3-7趟，5米/趟
    膝盖可以碰地，普通爬也OK
    ⏱️ ~10min（含组间休息）
  5. 🤸 拉伸放松 — 手臂伸展15秒×2 + 站姿体前屈15秒×2 + 大腿前侧15秒×2
    ⏱️ ~5min
  6. 💬 机动时间 — 补水、聊天、调整状态
    ⏱️ ~3min
⚡ 周五 · 速度+敏捷~45分钟
  1. 🏃 操场慢跑1圈（5分钟）+ 甩臂10次 + 转踝各10圈 + 小弹跳10次
    ⏱️ 热身 ~5min
  2. 🦵 高抬腿跑 — 15-30秒×2-3组，组间休息30秒
    足球脚尖快速点球，像鼓手敲鼓🥁
    慢速高抬腿走，不要求高度
    ⏱️ ~8min
  3. 🦘 立定跳远 — 8-12次（每次休息15秒）
    锥标记跳远(1m/1.2m/1.5m三档) 或 格子精准着陆跳(3分制)
    ⏱️ ~10min
  4. 🐍 绕锥蛇形跑 — 来回2-4趟，5锥间距1.5米
    增加"锥方向反应冲刺"：爸爸喊左/右/前，兜兜冲向对应锥
    慢速跑过去，走回来
    ⏱️ ~12min（蛇形跑+反应冲刺）
  5. 🤸 拉伸放松 — 同周一
    ⏱️ ~5min
  6. 💬 机动时间 — 补水、聊天、调整状态
    ⏱️ ~5min
💪 周六 · 力量+核心+闯关~45分钟
  1. 🏃 操场慢跑1圈（5分钟）+ 甩臂10次 + 转踝各10圈 + 小弹跳10次
    ⏱️ 热身 ~5min
  2. 🧱 平板支撑 — 15-45秒×2-3组，组间休息30秒
    平板传球：左右手交替从身下传网球
    膝盖着地半平板(跪姿)，5-10秒
    ⏱️ ~8min
  3. 🦘 夹球兔子跳 — 来回2-4趟，10米/趟
    可以用大一点的球(足球)
    ⏱️ ~10min
  4. 🏆 趣味障碍闯关 — 3-4轮，记录每轮用时
    闯关流程：高抬腿5米 → 卷腹5个 → 熊爬5米 → 单脚站10秒 → 侧步5米 → 跳远2次 → 完成
    道具升级赛道：蛇形绕锥 → 跳房子 → 精准着陆 → 方向冲刺 → 运球回来 → 抛接球5次
    ⏱️ ~15min（含轮间休息）
  5. 🤸 拉伸放松 — 同周一
    ⏱️ ~5min
  6. 💬 机动时间 — 补水、聊天、调整状态
    ⏱️ ~2min
🎾 周日 · 可选趣味（想练才练）~20-45分钟（看心情）
  1. 🟩 格子跳房子 — 来回3-5趟
  2. 🎾 网球抛接挑战 — 选喜欢的等级玩
  3. ⚽ 格子踢球射门 — 3米外踢球入格，比分数
  4. 🤸 拉伸放松 — 同周一
💡 周日纯玩性质，没有达标要求，重点是让孩子觉得运动=好玩。

⚠️ 爸爸必读安全守则

🛡️ 安全铁律

  1. 不在硬水泥地做跳跃动作 — 草地/瑜伽垫/木地板才可以
  2. 孩子说不舒服或疼痛，立刻停止 — 不要鼓励"坚持一下"
  3. 训练前确认无感冒发烧，饭后至少1小时
  4. 必须穿运动鞋 — 不穿拖鞋、不光脚
  5. 全程家长陪同 — 观察姿势，发现错误立即纠正
  6. 每组之间喝几口水 — 夏天早上也要注意补水
  7. 太热就改到傍晚或室内 — 不要硬撑高温
  8. 7岁骨骼还在发育，禁止负重训练 — 不拿哑铃、不背重物
  9. 出门前涂防晒 — 夏天早上8点紫外线已经很强
  10. 识别中暑信号 — 头晕、恶心、脸通红 → 立刻停，到阴凉处小口喝水
  11. 地面温度检查 — 训练前用手摸一下操场地面，烫手就改室内
